# Projeto 1 - Trocando valor de variáveis

### Inverter o valor de duas variáveis

```jsx
let a = 'vermelho'
let b = 'azul'

console.log(a)
//console: vermelho
console.log(b)
//console: azul
//---------------------------------------------------------------------------

let aux = a
a = b
b = aux

console.log(a)
//console: azul
console.log(b)
//console: vermelho
```
